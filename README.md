Recycling Tracker is a **personal project** designed to manage and track recycling activities in a simple, structured, and efficient way.  
The application allows registering people, managing recycling collections, storing historical data, and visualizing information through a lightweight web interface.

This project was built as a **learning-oriented and portfolio-ready application**, focusing on backend development in **C++**, database design, and local orchestration using **Docker Compose**.
